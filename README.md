# Path Finder Code Challenge FE Solution

This code works through the use of

- **ReactJs**,
- **TypeScript**

## Installation

- Install dependencies . run: 

 ```bash
 $ npm install
 ```

## Running the app

```bash
$ npm start
```

## App Running 

Go to `http://localhost:5000/` and you should see the project running.
